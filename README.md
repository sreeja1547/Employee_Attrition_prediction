# Employee Attrition Prediction using Machine Learning

## Project Overview
This project predicts whether an employee is likely to leave the company using machine learning. 
It helps HR teams identify at-risk employees and take preventive actions.

## Dataset
IBM HR Analytics Employee Attrition Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Imbalanced-learn
- Joblib

## Approach
- Data preprocessing (imputation, scaling, encoding)
- Handling class imbalance using SMOTE
- Logistic Regression model
- Hyperparameter tuning using GridSearchCV
- Threshold tuning for recall optimization

## Model Performance
- Attrition Recall: ~60%
- Accuracy: ~70%

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python src/train_model.py
